1. ОБЩИЕ ПОЛОЖЕНИЯ
________________________________________________________________________________

1.1. Настоящая Политика конфиденциальности (далее — «Политика») определяет
     порядок сбора, хранения, использования и защиты персональных данных
     пользователей мини-приложения «Шоколадница», размещённого в социальной
     сети ВКонтакте (далее — «Приложение»).

1.2. Оператором персональных данных является кафе «Шоколадница», расположенное
     по адресу: ул. Спасская, 18 (далее — «Оператор»).

1.3. Используя Приложение, Пользователь выражает безоговорочное согласие с
     настоящей Политикой. В случае несогласия с условиями Политики Пользователь
     обязан прекратить использование Приложения.

1.4. Настоящая Политика разработана в соответствии с:
     — Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных»;
     — Федеральным законом от 27.07.2006 № 149-ФЗ «Об информации,
       информационных технологиях и о защите информации»;
     — Постановлением Правительства РФ от 01.11.2012 № 1119
       «Об утверждении требований к защите персональных данных»;
     — ГОСТ Р 7.0.97-2016 «Система стандартов по информации, библиотечному
       и издательскому делу».

________________________________________________________________________________

2. СОСТАВ СОБИРАЕМЫХ ПЕРСОНАЛЬНЫХ ДАННЫХ
________________________________________________________________________________

2.1. При оформлении бронирования столика Оператор собирает следующие
     персональные данные Пользователя:

     а) Фамилия и имя (обязательное поле);
     б) Номер мобильного телефона (обязательное поле);
     в) Дата и время планируемого визита (обязательное поле);
     г) Количество гостей (обязательное поле);
     д) Комментарий к бронированию (необязательное поле, свободный текст);
     е) Идентификатор пользователя ВКонтакте (VK user ID) — только при
        добровольном предоставлении согласия на получение уведомлений;
     ж) Согласие на получение уведомлений от сообщества ВКонтакте (флаг).

2.2. В целях обеспечения безопасности и предотвращения злоупотреблений
     Оператор также обрабатывает:
     — IP-адрес устройства Пользователя (хранится не более 120 секунд
       в базе данных, используется исключительно для ограничения частоты
       запросов);
     — технические журналы ошибок (сообщения об ошибках, трассировки стека,
       без явного указания имени или телефона).

2.3. Оператор не собирает и не обрабатывает:
     — паспортные данные;
     — данные банковских карт;
     — биометрические данные;
     — данные о местоположении (геолокацию);
     — файлы cookie и данные локального хранилища браузера.

________________________________________________________________________________

3. ЦЕЛИ ОБРАБОТКИ ПЕРСОНАЛЬНЫХ ДАННЫХ
________________________________________________________________________________

3.1. Персональные данные обрабатываются исключительно в следующих целях:

     а) Оформление и подтверждение бронирования столика в кафе;
     б) Информирование персонала кафе о предстоящих бронированиях;
     в) Направление Пользователю напоминания о визите (за 1 час до времени
        бронирования) — только при наличии согласия Пользователя;
     г) Направление Пользователю запроса обратной связи (на следующий день
        после визита) — только при наличии согласия Пользователя;
     д) Ведение статистики посещаемости в обезличенной форме;
     е) Обеспечение безопасности информационной системы и предотвращение
        злоупотреблений.

3.2. Оператор не использует персональные данные для:
     — передачи третьим лицам в коммерческих целях;
     — таргетированной рекламы;
     — создания профилей поведения Пользователя;
     — автоматизированного принятия решений, существенно влияющих на права
       Пользователя.

4. ПРАВОВЫЕ ОСНОВАНИЯ ОБРАБОТКИ
________________________________________________________________________________

4.1. Обработка персональных данных осуществляется на следующих правовых
     основаниях:

     а) Согласие субъекта персональных данных (ст. 6, ч. 1, п. 1 Закона
        № 152-ФЗ) — основное основание для сбора данных при бронировании;
     б) Исполнение договора (ст. 6, ч. 1, п. 5 Закона № 152-ФЗ) — обработка
        данных, необходимых для оказания услуги бронирования столика;
     в) Законный интерес Оператора (ст. 6, ч. 1, п. 7 Закона № 152-ФЗ) —
        ведение журналов безопасности и предотвращение злоупотреблений.

________________________________________________________________________________

5. ПЕРЕДАЧА ПЕРСОНАЛЬНЫХ ДАННЫХ ТРЕТЬИМ ЛИЦАМ
________________________________________________________________________________

5.1. Персональные данные могут быть переданы следующим получателям:

     а) Персонал кафе (официанты, администраторы) — получают сведения об
        имени гостя, дате, времени, количестве гостей и номере телефона через
        защищённый мессенджер ВКонтакте в целях подготовки к визиту;
     б) ВКонтакте (VK.com) — платформа-посредник, через API которой
        направляются уведомления Пользователю и персоналу. Обработка данных
        в рамках платформы регулируется Политикой конфиденциальности ВКонтакте
        (https://vk.com/privacy);
     в) Sentry (sentry.io) — при технических ошибках, если Оператором
        активирован соответствующий сервис мониторинга. Обрабатываются
        технические данные (трассировки ошибок). Предоставление данного сервиса
        является опциональным.

5.2. Оператор не передаёт персональные данные:
     — государственным органам, за исключением случаев, предусмотренных
       действующим законодательством Российской Федерации;
     — иным третьим лицам без явного согласия Пользователя.

5.3. При передаче персональных данных через API ВКонтакте применяются
     технические меры защиты (TLS/HTTPS), обеспечивающие конфиденциальность
     передаваемых сведений.

________________________________________________________________________________

6. СРОКИ ХРАНЕНИЯ ПЕРСОНАЛЬНЫХ ДАННЫХ
________________________________________________________________________________

6.1. Данные бронирований (имя, телефон, дата, время, количество гостей,
     комментарий, идентификатор ВКонтакте) хранятся в базе данных Оператора
     на протяжении всего периода деятельности кафе либо до получения
     Оператором требования об удалении.

6.2. IP-адреса в журналах ограничения запросов хранятся не более 120 секунд
     с момента записи и удаляются автоматически.

6.3. Технические журналы ошибок хранятся бессрочно в целях расследования
     инцидентов безопасности.

6.4. Запланированные задачи (напоминания, запросы обратной связи) хранятся
     до момента исполнения или отмены бронирования.

8. МЕРЫ ЗАЩИТЫ ПЕРСОНАЛЬНЫХ ДАННЫХ
________________________________________________________________________________

8.1. Оператор принимает следующие технические и организационные меры для
     защиты персональных данных:

     — Шифрование данных при передаче по протоколам TLS/HTTPS;
     — Хранение данных в СУБД PostgreSQL с ограниченным доступом;
     — Разграничение прав доступа к данным (персонал кафе видит только
       необходимые для работы поля);
     — Ограничение частоты запросов (rate limiting) для предотвращения
       несанкционированного доступа;
     — Мониторинг ошибок и аномалий в информационной системе;
     — Заголовки безопасности HTTP (Content-Security-Policy, X-Frame-Options
       и др.) применяются ко всем API-ответам.

8.2. Доступ к персональным данным имеют исключительно уполномоченные
     сотрудники кафе, принявшие обязательства о конфиденциальности.
