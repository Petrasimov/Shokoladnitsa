## ⚙️ Настройка

### Шаг 1: Включите Long Poll в группе VK

1. Откройте вашу группу VK → **Управление** → **Работа с API** → **Long Poll API**
2. Включите **Long Poll API**
3. **Версия API:** `5.199`
4. **Типы событий:** отметьте `Входящие сообщения`

### Шаг 2: Включите сообщения сообщества

1. В группе: **Управление** → **Сообщения** → **Настройки для бота**
2. Включите **Возможности ботов**
3. Разрешите **Добавление в беседы**

### Шаг 3: Получите ID беседы с официантами

**Способ 1 - через браузер:**
1. Откройте беседу в VK
2. Посмотрите URL: `vk.com/im?sel=c123`
3. Число после `c` - это ID беседы
4. **Важно:** В .env нужно указать `peer_id = 2000000000 + ID беседы`
5. Пример: если ID=123, то `VK_WAITERS_CHAT_ID=2000000123`

**Способ 2 - добавьте бота в беседу и он покажет peer_id:**
1. Добавьте сообщество в беседу
2. Напишите любое сообщение
3. Бот получит peer_id автоматически (см. логи)

### Шаг 4: Заполните .env

Откройте `backend/.env` и укажите:

```env
VK_COMMUNITY_TOKEN=ваш_токен_сообщества
VK_GROUP_ID=234068981
VK_ADMIN_ID=ваш_vk_id
VK_WAITERS_CHAT_ID=2000000123
CAFE_ADDRESS=ул.Чапаева 18
```

**Где взять:**
- `VK_COMMUNITY_TOKEN` - Управление → Работа с API → Ключи доступа → Создать ключ (нужны права на сообщения)
- `VK_GROUP_ID` - ID вашей группы (см. адрес страницы)
- `VK_ADMIN_ID` - ваш VK ID (см. MIGRATION_TELEGRAM_TO_VK.md)
- `VK_WAITERS_CHAT_ID` - peer_id беседы (2000000000 + ID беседы)

## 🚀 Запуск

### Вариант 1: Запуск всего вместе (рекомендуется)

```bash
cd backend
python start_all.py
```

Это запустит:
- FastAPI сервер на порту 8000
- VK Bot Long Poll

### Вариант 2: Раздельный запуск

**Терминал 1 - FastAPI:**
```bash
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**Терминал 2 - VK Bot:**
```bash
cd backend
python -m app.vk_bot_server
```

## 📱 Использование

### Для официантов (в групповой беседе)

Когда приходит новая бронь:
```
📌 НОВАЯ БРОНЬ

👤 Иван Петров
👥 Гостей: 4
📅 2026-02-15 в 19:00
📞 +79001234567

🆔 Бронь #123
```

В момент визита гостя:
```
⏰ ГОСТЬ ДОЛЖЕН ПРИЙТИ

👤 Иван Петров
👥 Гостей: 4
🕒 Время: 19:00

Гость пришёл?
[✅ Пришёл] [❌ Не пришёл]
```

**Если нажать "✅ Пришёл":**
- Бот попросит ввести сумму чека
- Введите число, например: `2500`
- Бот сохранит: гость пришёл, чек 2500₽

**Если нажать "❌ Не пришёл":**
- Бот отметит: гость не пришёл

### Для администратора (в ЛС с ботом)

1. Напишите боту: `начать` или `/start`
2. Откроется меню с кнопками:

```
👋 Админ-панель

Выберите действие:
[📊 Статистика] [📥 Экспорт CSV]
[📈 Графики]
```

**Статистика:**
```
📊 Статистика бронирований

📌 Всего броней: 145
👥 Всего гостей: 482
👥 Среднее гостей: 3.3
✅ Пришли: 128
❌ Не пришли: 17
```

**Экспорт CSV:**
- Отправит файл `reservations.csv` со всеми бронями

**Графики:**
- Выберите график → бот отправит изображение

## ❓ Troubleshooting

### Бот не отвечает в ЛС

1. Проверьте `VK_ADMIN_ID` в `.env`
2. Напишите боту `начать` - он покажет клавиатуру
3. Проверьте логи: `python -m app.vk_bot_server`

### Не приходят сообщения в группу официантов

1. Проверьте `VK_WAITERS_CHAT_ID` (должен начинаться с `2000000`)
2. Убедитесь что бот добавлен в беседу
3. Проверьте права токена (должны быть права на сообщения)

### Кнопки не работают

1. Убедитесь что Long Poll включен в группе
2. Проверьте версию API: должна быть `5.199`
3. Включите события "Входящие сообщения"

### Графики не отправляются

1. Проверьте что matplotlib установлен: `pip install matplotlib`
2. Проверьте права доступа к файлам (папка `backend/app/`)
3. Смотрите ошибки в логах
